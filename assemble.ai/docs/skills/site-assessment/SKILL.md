# Skill: Site Assessment

**Tier 1 — No scaffold required.** The Feasibility Agent assesses site characteristics using its training knowledge, web research, and any provided documents. This skill is a reference document.

## When This Skill Applies

- Assessing a new development site
- Reviewing site characteristics, dimensions, services, access
- Identifying site constraints and opportunities
- Answering "what do we know about this site?"

## What the Feasibility Agent Does

1. **Identify** the LGA (council) from the site address
2. **Research** using external sources: planning portals, council websites, flood maps, heritage registers, EPA contamination register
3. **Review** any provided documents: title search, survey, prior reports
4. **Compile** the site summary in standard format
5. **Flag** information gaps with recommended actions

## Site Assessment Checklist

Work through these factors for every site:

**Location and Context**
- [ ] Suburb profile and demographics
- [ ] Proximity to transport, amenity, services
- [ ] Neighbourhood character — current and likely future
- [ ] Comparable sales or rentals in the area (if residential)

**Physical Site**
- [ ] Site area (m²)
- [ ] Street frontage (m)
- [ ] Depth (m)
- [ ] Shape — regular/irregular, battle-axe, flag
- [ ] Orientation — compass direction of street frontage
- [ ] Topography — slope, natural ground levels, fall direction
- [ ] Existing improvements — buildings, structures, trees
- [ ] Heritage listing on site or adjacent

**Services**
- [ ] Sewer — connected, location of sewer main
- [ ] Water — connected, location of water main
- [ ] Electricity — connected, capacity adequate for intended use
- [ ] Gas — connected/available
- [ ] Telecommunications — available

**Access**
- [ ] Street frontage access
- [ ] Rear lane or secondary access
- [ ] Vehicle access constraints (crossover, kerb)
- [ ] Service access

**Easements and Restrictions**
- [ ] Drainage easements
- [ ] Rights of way
- [ ] Restrictive covenants
- [ ] 88B restrictions on the title
- [ ] Tree preservation orders (TPO)

**Environmental Flags**
- [ ] Flood mapping status
- [ ] Bushfire prone land
- [ ] Heritage listing — site, adjacent, conservation area
- [ ] Contamination history (site use, EPA register)
- [ ] Biodiversity mapping

## Site Summary Format

```
SITE SUMMARY
Address:      [Full address]
Lot/DP:       [Lot X DP XXXXXXX (and Lot Y DP XXXXXXX if multiple lots)]
LGA:          [Council name]
Area:         [X,XXX]m²
Frontage:     [X.X]m to [street name]
Depth:        [~X.X]m (varies — [note if irregular])
Orientation:  [North/South/East/West]-facing to [street name]
Topography:   [Generally flat / slight fall X.Xm to [direction] / steep fall etc.]
Existing:     [Description of current buildings, trees, structures. Heritage note.]
Services:     [All services connected. OR Note specific gaps.]
Access:       [Description — frontage access, any rear lane, constraints]
Easements:    [Describe or "None identified on available information"]

KEY OBSERVATIONS:
+ [Positive 1 — e.g., north-facing frontage]
+ [Positive 2 — e.g., flat site, minimal excavation]
+ [Positive 3 — e.g., rear lane access]
- [Negative/risk 1 — e.g., drainage easement constrains setback]
- [Negative/risk 2 — e.g., adjoining residential to south]
- [Negative/risk 3 — e.g., probable asbestos in existing structure]
```

## Information Gaps

For each gap, state clearly and recommend action:

```
⚠ NOT AVAILABLE — requires action:
- [Item]: [Why it matters] → [Recommended action and indicative cost/timeframe]
```

Common gaps:
- Title search not provided → Cannot confirm easements, covenants, restrictions
- No survey → Exact dimensions and levels unknown
- No geotech → Soil and rock conditions unknown
- AHIMS not searched → Aboriginal heritage status not confirmed
- No contamination assessment → Industrial history not investigated
