# Quickstart: Intelligent Document Repository

## Prerequisites
- Node.js 20+
- npm / pnpm

## Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Initialize Database**
   ```bash
   npm run db:push
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```

4. **Verify Feature**
   - Open `http://localhost:3000/documents`
   - Drag and drop a PDF file.
   - Verify it appears in the list.
